# To-Do List REST API на C++

Полнофункциональный REST API для управления списком задач, реализованный на C++ с использованием собственного HTTP сервера.

## Условие задачи

**Задание:** Создание API (Rest) для управления списком задач (To-Do List), включающий эндпоинты для CRUD-операций (Create, Read, Update, Delete) с использованием методов GET, POST, PUT/PATCH, DELETE, работающих с ресурсами типа task.

**Требования:**
- Реализация CRUD операций
- Использование стандартных HTTP-методов
- Применение корректных HTTP-кодов (200, 201, 404)
- Формат обмена данными — JSON
- Дополнительные функции: логирование, кэширование

## Ключевые особенности

###Основной функционал
- **Полный CRUD** для задач (Create, Read, Update, Delete)
- **6 RESTful эндпоинтов** с поддержкой HTTP методов
- **JSON форматы** для запросов и ответов
- **Правильные HTTP коды** (200, 201, 400, 404, 500)
